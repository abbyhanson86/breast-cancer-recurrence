# breast-cancer-recurrence
In our project, we set out to predict if a breast cancer patient will have a recurrence based on various factors such as age, menopause status, tumor size, cancer stage, and other factors. 

In this project, we used two datasets: one from the University of California Irvine, one from the German Breast Cancer Study Group.

* [UCI dataset](https://archive-beta.ics.uci.edu/dataset/14/breast+cancer)
* [German dataset](https://www.kaggle.com/datasets/utkarshx27/breast-cancer-dataset-used-royston-and-altman)
